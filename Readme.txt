The only prerequisite for this application to run is to have nodejs and npm installed on your machine.
This can be downloaded from this link: https://nodejs.org/en/download/
This will install both nodejs and npm on your machine.

1. Extract zip file "immedia-assesment.zip",
2. Open extracted folder
3. Open command line interface in that folder by holding in "Shift" button and right-clicking in the folder and select  "Open PowerShell window here"/"Open Command Prompt window here" from the context menu.
4. In the command line interface type the following "npm install" to install all dependencies wait for it all to install.
5. Next type "npm run serve" to run the application
6. Open browser on "localhost:8080"
